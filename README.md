# AI CV Analyzer

## Overview

AI CV Analyzer is a Python and Streamlit application that compares a candidate's CV against a job description using Natural Language Processing (NLP) techniques.

The system calculates a similarity score and identifies keywords that may be missing from the CV, helping users improve their chances of matching job requirements.

## Features

* Compare CVs with job descriptions
* Calculate similarity scores using TF-IDF and Cosine Similarity
* Identify missing keywords
* Interactive Streamlit interface
* Real-time analysis

## Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* Natural Language Processing (NLP)

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Tumi-Manamela/ai-cv-analyzer.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
